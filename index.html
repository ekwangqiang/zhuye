我的技能

优逸客公司的老师教导我们的技能

  • HTML

    超文本标记语言HyperText Markup Language(简称:HTML);标准通用标记语言下的一个应用，在浏览器中解析执行

  • CSS

    层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML(标准通用标记语言的一个应用)

  • JavaScript

    属于网络的脚本语言! JavaScript 被数百万计的网页用来改进设计、验证表单、检测浏览器、创建cookies,以及更多的应用。

  • PHP

    是一种创建动态交互性站点的强有力的服务器端脚本语言（英文全称名:Hypertext Preprocessor)是一种通用开源脚本语言。

  • React

    是一个用于构建用户界面的 JAVASCRIPT 库。 React主要用于构建UI,很多人认为 React 是 MVC 中的 V(视图)。

  • Vue.js

    是用于构建交互式的 Web 界面的库。它提供了 MVVM 数据绑定和一个可组合的组件系统,具有简单、灵活的 API。
